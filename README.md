# Описание `seller.py`

Этот скрипт помогает продавцу управлять своими товарами на маркетплейсе и контролировать их характеристики.

Скрипт умеет:
 - Получяет список всех товаров продавца
 - Загружать список товаров магазина Ozon
 - Скачивать актуальные остатки с сайта Casio
 - Обновлять остатки в Ozon
 - Обновлять цены в Ozon

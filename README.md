# Описание `seller.py`

Этот скрипт помогает продавцу управлять своими товарами на маркетплейсе и контролировать их характеристики.

Скрипт умеет:
 - Получяет список всех товаров продавца
 - Загружать список товаров магазина Ozon
 - Скачивать актуальные остатки с сайта Casio
 - Обновлять остатки в Ozon
 - Обновлять цены в Ozon

# Описание `market.py`

Этот скрипт предназначен для продавцов маркетплейса и помогает автоматически собирать и анализировать информацию о ценах на товары

Скрипто умеет:
 - Пполучяет список всех товаров продавца
 - Определять фактические цены, которые видят покупатели
 - Для каждой кампании (FBS и DBS):
    - Получает список товаров
    - Обновляет остатки
    - Обновляет цены
